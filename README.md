Sambit Kumar Behera's Personal Portfolio
This is a single-page portfolio website showcasing my professional experience, skills, and projects. It is built using modern web technologies to be fast, responsive, and visually appealing.

About
This portfolio serves as a central hub for my professional profile. It includes a detailed "About Me" section, highlights key projects from my work at Fiery Digital Imaging and Samsung Research, and provides a comprehensive list of my technical skills.

Features
Responsive Design: Adapts to desktop, tablet, and mobile screens.

Typing Animation: An engaging animated tagline on the hero section.

Dynamic Sections: Dedicated sections for About, Projects, Skills, and Contact.

One-Click Copy: Easily copy my email address to your clipboard.

Getting Started
To get a local copy up and running, follow these simple steps.

Prerequisites
A modern web browser.

Installation
Clone the repository:

git clone [https://github.com/your-username/your-repo-name.git](https://github.com/sam-beat/sambit-portfolio.git)

Navigate to the project directory:

cd your-repo-name

Open the file:
Simply open the portfolio.html file in your preferred web browser to view the website.

Tech Stack
HTML: For the structure of the page.

Tailwind CSS: For all styling, providing a utility-first approach.

JavaScript: For interactivity, including the typing animation and copy-to-clipboard functionality.
